# WrightCapBot

**WrightCapBot** (ausgesprochen "Right Cap Bot") ist ein KI-gestuetzter Praxis-Assistent
fuer bayerische Arztpraxen und medizinische Einrichtungen.

## Inhalt

- `/extensions/` — Fertige Extensions fuer Endanwender (Arztbrief-Schreiber, QM, etc.)
- `/distributor/` — Extensions Distributor / Repo-Katalog (Dashboard-Komponente)
- `extension.json` — Extension-Manifest

## Installation

1. Repository klonen: `git clone https://github.com/jarvix-code/wrightcapbot.git`
2. Dashboard starten (siehe Dokumentation)
3. Extensions ueber den Repo-Katalog im Dashboard laden
